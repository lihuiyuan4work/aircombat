## 躲避方块小游戏

用浏览器玩的简单小游戏，你控制底部的小方块，左右移动来躲避从上方掉落的障碍物，尽量存活更久、获得更高分数。

### 玩法说明

- **操作**：键盘 ← → 或 A / D 控制左右移动  
- **计分**：每成功躲过一批障碍会加分  
- **结束**：被障碍物撞到则游戏结束  
- **重开**：游戏结束后按空格键重新开始

### 运行方式

有两种方式：

1. **直接用浏览器打开**
   - 在文件管理器中打开 `mini-game` 文件夹
   - 双击 `index.html`，用浏览器打开即可开始游戏

2. **用命令行启动本地服务（需要已安装 Node.js）**
   - 打开终端，切到此目录：
     - `cd C:\Users\honor\mini-game`
   - 运行：
     - `npm run start`
   - 终端里会提示本地访问地址（一般是 `http://localhost:3000` 或 `http://localhost:5000` 之类），
     在浏览器里打开即可玩。
